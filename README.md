## Service Requests Management Power BI Dashboard Dataset

This guide provides a dataset to help you practice building the Service Requests Management Power BI Dashboard. Use this dataset to recreate the dashboard and customize visuals, measures, and filters based on your own service operations.
Why This Template

This template is designed as a ready-to-use Power BI Dashboard suitable for business or personal reporting needs. All dashboard elements are editable directly inside Microsoft Power BI. You can easily replace the data, adjust colors, and add your own branding.
Dashboard Contents & Analysis

    Total number of service requests received
    Service requests by month with average daily requests
    Status distribution: completed, pending, and cancelled
    Total paid amount and paid amount by month
    Comparison between paid amounts and quotations
    Service type analysis based on counts and revenue
    Total paid amount by regions with geographic map visualization
    Monthly filter for detailed performance analysis

Key Features

    Analysis covering more than 15 data perspectives
    Wide screen layout optimized for Full HD displays
    Preview images included for reference
    Free fonts and icons included
    Fully editable visuals and data models

Package Includes

    Power BI file (.PBIX) – approximately 1 MB
    Three Excel data tables used as the dashboard data source
